How to use this library

  1. Pick the agent(s) that match your client’s workflow (settlement, exceptions, liquidity, governance, exec reporting).
  2. Use the Phase 2 Playbooks to scope a pilot (Discovery → SOW → Pilot → Runbook).
  3. Use the agent’s KPIs and boundaries to set expectations and avoid risky claims.
  4. For implementation details, follow certification and partner tier rules (see IP Notice).
