Scoring

Deployment Readiness Scoring

Score Data, Governance, and Stakeholder Alignment. Two Reds = no deployment.

Install Home Integration Checklist Operator Install

1) Data Readiness

  • Green: feeds available, consistent schema, history available
  • Yellow: partial feeds, schema drift, limited history
  • Red: no access, unclear ownership, unreliable exports

2) Governance Readiness

  • Green: access controls + logging + change control defined
  • Yellow: some controls, missing incident path
  • Red: no logging expectations, no owners, no boundaries

3) Stakeholder Alignment

  • Green: exec sponsor + ops owner + cadence agreed
  • Yellow: sponsor exists, missing cadence or KPI baseline
  • Red: no owner, no timeline, “try it and see”

Decision rule

Two Reds = stop and return to Discovery. One Red = proceed only with remediation plan + sponsor sign-off.
