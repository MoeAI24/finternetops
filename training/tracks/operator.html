Track A • Operator

Operator Track

For technical implementers operating MBCC agents in production. Wire data sources, validate outputs, triage exceptions, and produce daily ops reporting without creating compliance risk.

Open Onboarding Start Foundations Dashboard

Requirements

  • Understands tokenized asset flows and operational risk points (settlement, exceptions, reconciliation).
  • Can run event pipeline inputs (API/webhook/polling) into dashboards + alerts.
  • Enforces guardrails: human-in-loop, citations, and no fabricated outputs.
  • Produces runbooks, logs, and KPIs for client acceptance.

Deliverables you must be able to produce

  • Deployment checklist + runbook for at least 3 agents
  • Daily exception triage report (1-page)
  • Alert thresholds + escalation matrix
  • Validation notes + acceptance criteria

Recommended learning path

  1. Foundations → pass quiz
  2. Complete your track’s core modules
  3. Use Partner Toolkit templates to build a pilot package
  4. Deliver a mock executive readout (15 minutes)
  5. Take certification exam (80% pass)
